### 실습해본 _JAVA_ code

```java

public class Calculator {
  public static void main(String[] args){
    int a = 4;
    int b = 2;
    System.out.println(sum(a,b));
    System.out.println(minus(a,b));
  }
  private static int sum(int a,int b){
    return a+b;
  }
  private static int minus(int a,int b){
    return a-b;
  }
}
```
